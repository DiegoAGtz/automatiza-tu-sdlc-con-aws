## Hola, mundo DevOps

Este es el primer post de nuestro blog de demostración. El propósito de este sitio es mostrar cómo funciona un pipeline de CI/CD completo en AWS.

### Qué vas a encontrar aquí

- **Guías prácticas** sobre automatización con AWS
- **Ejemplos reales** de infraestructura como código
- **Mejores prácticas** de DevOps y CI/CD

### El stack tecnológico

| Componente | Tecnología |
|-----------|-----------|
| Generador | Hugo |
| Hosting | Amazon S3 |
| CDN | CloudFront |
| IaC | AWS CDK (TypeScript) |
| CI/CD | CodePipeline + CodeBuild |

Todo el código de este proyecto es open source y está disponible para que lo explores y lo adaptes a tus necesidades.

> "La automatización no es un lujo, es una necesidad." - Todo equipo DevOps, siempre.
